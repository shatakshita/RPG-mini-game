# RPG Mini Game

**Interactive Python program to create your own RPG character with Strength, Intelligence, and Charisma, featuring humor, hints, and special archetypes.**

---

## Description

This is a fun and interactive Python program that lets you design your RPG character. Enter a name and stats, and the program will:

- Validate your inputs with clear and funny messages  
- Display stats visually using dots (● filled, ○ empty)  
- Detect special archetypes:  
  - **Specialist:** Focused on one stat  
  - **Balanced:** Evenly distributed stats  
  - **Wildcard:** Unique combinations  
- Give hints and playful feedback if your combination isn’t perfect  
- Challenge players to find hidden “perfect” stats  

It’s a simple, text-based Python program designed to be engaging and humorous — perfect for learning Python while having fun!

---

## Features

- Interactive input for **Name**, **Strength**, **Intelligence**, and **Charisma**  
- Input validation for correct data types and ranges  
- Visual stat bars using filled and empty dots  
- Detection of legendary archetypes  
- Hints for clever players  
- Humorous feedback for fun
